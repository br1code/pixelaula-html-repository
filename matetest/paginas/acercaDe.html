Informacion del Proyecto y Licencias


Desarrollado utilizando los lenguajes HTML, Java Script Y PHP.

En este proyecto se ha utilizado Bootstrap con la licencia "MIT".

El objetivo de este proyecto es proveer a alumnos de nivel primario una herramienta de practica facil de adaptar a multiples temas del area de matematicas. El objetivo del juego sera responder la mayor cantidad de veces posibles la ecuacion o cuenta ofrecida en el menor periodo de tiempo posible dentro de un intervalo de 30 segundos. Lo importante aqui no es la capacidad de los alumnos de realizar calculos mentales sino su capacidad de comprender el ejercicio dado y aplicar la formula correcta en el menor tiempo posible, por lo que se les provee una calculadora. Si usted presiona el boton "iniciar" en esta pantalla se le mostraran ejemplos aleatorios de todos los temas disponibles.
